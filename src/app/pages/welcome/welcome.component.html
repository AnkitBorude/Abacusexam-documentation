Abacus Exam Application

REST API Documentation

Built by Ankit Borude

The Abacus Exam Web API is a comprehensive backend solution designed to streamline the management and evaluation of Abacus-based exams. This RESTful API provides a robust, scalable, and secure platform for handling exam-related operations, enabling developers to integrate its functionality into various front-end applications or third-party systems.

Technologies Used

Developement Tools

Key Features

Authorization

Implemented Authorization and RBAC using JWT Tokens

Data Validation

Implemented data validation and sanitization using the Joi npm package.

Logging and Monitoring

Tracked API usage, errors, and performance using the Winston and Morgan npm packages.

Question

Automatic question generation and resolution for exams with MCQs and answers
