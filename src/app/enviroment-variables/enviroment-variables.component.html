Production Enviroment Variables (For Docker Only)

The .prod.env file contains production-specific environment variables. This file and its variables are only required for the Docker-specific production installation and not for normal installation or docker compose local deployment, where the .env file is sufficient for both local and production deployment. While deploying to productionthrough Docker Compose, the given .prod.env file must also be created alongside the .env file in the root folder of the project.
